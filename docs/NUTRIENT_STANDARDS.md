# tt-ni 영양성분 분석 및 약물 상호작용 판정 기준

이 문서는 **tt-ni** 서비스의 분석 엔진이 영양소 섭취 상태를 판정하고, 복용 약물 간 상호작용을 검사하며, 복용 스케줄을 최적화할 때 사용하는 과학적·의학적 기준을 명세합니다.

---

## 1. 영양소 섭취 상태 판정 기준 (KDRIs 2025)

tt-ni의 영양 분석 엔진은 **2025년 한국인 영양소 섭취기준 (KDRIs: Dietary Reference Intakes for Koreans)**을 기준으로 설계되었습니다. 사용자의 프로필(성별, 출생연도, 키, 몸무게, 기저질환, 임신·수유 여부)을 반영하여 매핑된 개인별 맞춤 기준치를 토대로 총 섭취량을 판정합니다.

### 섭취 상태 정의 및 판정 로직

영양소별 1일 섭취 총량(식품을 제외한 영양제 조합 기준)은 아래의 네 가지 기준 구간에 의해 모니터링됩니다.

```
[ 부족 (deficient) ] <--- (RDA 또는 AI) ---> [ 적정 (normal) ] <--- (UL) ---> [ 과다/초과 (caution / excess) ]
```

* **RDA (Recommended Dietary Allowance, 권장섭취량)**: 인구 집단의 약 97~98%에 해당하는 사람들의 일일 영양소 요구량을 충족시키는 섭취 수준입니다.
* **AI (Adequate Intake, 충분섭취량)**: 권장섭취량을 산출하기 위한 과학적 근거가 부족한 경우, 건강한 사람들의 섭취량을 토대로 설정한 충분한 섭취 수준입니다. (예: 오메가-3, 비오틴, 식이섬유 등)
* **UL (Tolerable Upper Intake Level, 상한섭취량)**: 인체에 유해한 영향이 나타나지 않는 일일 최대 영양소 섭취 수준입니다. (예: 비타민 A, 비타민 D, 철분, 아연 등)

### 상태별 판정 세부 규칙

1. **부족 (deficient)**: 
   * 해당 영양소에 **RDA**가 존재할 경우: `일일 총섭취량 < RDA`
   * 해당 영양소에 **AI**만 존재할 경우: `일일 총섭취량 < AI`
   * *경고 발생*: 일반적인 건강 기능을 원활히 지원하기에 일일 섭취량이 부족한 상태임을 사용자에게 알립니다.
2. **적정 (normal)**:
   * `RDA (또는 AI) <= 일일 총섭취량 <= UL (존재할 경우)`
   * *알림*: 신체 기능 보조를 위한 완벽하고 안정적인 수준입니다.
3. **초과/과다 (caution / excess)**:
   * **주의 (caution)**: 상한섭취량(UL)이 있는 영양소에서 `UL < 일일 총섭취량 < UL * 1.5` 구간일 때 판정됩니다. 만성 질환 예방 또는 단기 고함량 섭취로 볼 수 있으나 장기 복용 시 주의를 요구합니다.
   * **위험/초과 (excess)**: `일일 총섭취량 >= UL * 1.5`이거나 상한섭취량이 존재하지 않는 일부 비타민에서 비정상적인 고용량 섭취가 감지되었을 때 판정됩니다.
   * *경고 발생*: 독성 유발 및 소화기 장애 등 부작용 우려가 있으므로 감량을 적극 권장합니다.

## 2. 약물-영양소 상호작용 (DNI: Drug-Nutrient Interaction) 판정 기준

복용 중인 전문/일반 의약품과 섭취하려는 영양 성분 간에 발생할 수 있는 흡수율 저하, 약효 감소, 또는 독성 증가 등의 상호작용을 차단하기 위한 15가지 핵심 임상 판정 규칙입니다.

| 약물 계열 | 영양소 성분 | 상호작용 기전 및 판정 결과 | 경고 등급 |
|:---|:---|:---|:---:|
| **와파린 (Warfarin)** | 비타민 K | 비타민 K가 와파린의 항응고(혈액 희석) 작용을 길항하여 혈전 생성 위험을 급격히 높입니다. 병용이 원천 금지됩니다. | **High (금기)** |
| **스타틴 계열 (Statins)** | Coenzyme Q10 | 스타틴계 약물이 체내 CoQ10 합성 경로를 차단하여 근육통(근병증) 부작용을 유발합니다. **CoQ10 적극 보충 권장**. | **Info (권장)** |
| **메트포르민 (Metformin)** | 비타민 B12 | 메트포르민의 장기 복용이 장내 비타민 B12 흡수를 저해하여 빈혈 및 신경병증을 유발할 수 있습니다. **B12 추가 섭취 권장**. | **Info (권장)** |
| **아스피린 (Aspirin)** | 은행잎 추출물 (Ginkgo) | 양쪽 성분 모두 강력한 혈액 응고 억제(항혈소판) 작용이 있어, 동시 복용 시 잇몸 출혈, 내출혈, 뇌출혈 위험이 증가합니다. | **High (금기)** |
| **루프 이뇨제 (Loop Diuretics)** | 칼륨 / 마그네슘 | 소변을 통한 칼륨과 마그네슘 배출을 촉진하여 저칼륨혈증이나 저마그네슘혈증(근육 경련 등)을 유발합니다. **추가 섭취 권장**. | **Medium (주의)** |
| **제산제 (Antacids)** | 철분 (Iron) / 칼슘 | 위산 분비를 억제하여 철분과 칼슘의 이온화 및 장내 흡수율을 크게 떨어뜨립니다. **최소 2시간 이상 시간차 복용 권장**. | **Medium (주의)** |
| **테트라사이클린계 항생제** | 칼슘 / 철분 / 마그네슘 | 항생제가 다가 금속 이온(Ca, Fe, Mg)과 킬레이트 결합을 형성하여 항생제의 장내 흡수를 차단, 항생제 약효를 소멸시킵니다. **최소 2시간 시간차 필수**. | **High (금기)** |
| **퀴놀론계 항생제** | 아연 / 철분 / 칼슘 | 테트라사이클린과 유사하게 금속 성분과 결합하여 항생제의 생체 이용률을 심각하게 감소시킵니다. **최소 2시간 시간차 필수**. | **High (금기)** |
| **갑상선 호르몬제 (Synthroid)** | 칼슘 / 철분 | 칼슘과 철분이 갑상선 호르몬제(레보티록신)의 장내 흡수를 방해하여 갑상선 기능 저하 상태가 개선되지 않게 만듭니다. **최소 4시간 시간차 강력 권장**. | **High (금기)** |
| **ACE 차단제 (혈압약)** | 고용량 칼륨 (Potassium) | 칼륨 배출을 억제하여 고칼륨혈증 및 부정맥 유발 위험이 있습니다. 칼륨 영양제와의 고농도 병용을 제안 및 경고합니다. | **Medium (주의)** |

## 3. 영양소 간 흡수 경쟁 및 시너지 판정 기준

동시에 장 내로 들어왔을 때 서로 흡수를 방해하는 **길항(Antagonism) 작용**과, 반대로 흡수 및 생체 이용 효율을 돕는 **시너지(Synergy) 작용**을 판별하여 최적의 섭취 시점을 조절합니다.

### 🚫 길항 작용 및 시간차 복용 기준 (Antagonism)
분석 엔진은 아래 조합이 감지될 경우, 동시 복용을 방지하고 복용 스케줄러에서 아침/저녁 등으로 섭취 시점을 원천 격리시킵니다.

* **칼슘 (Calcium) vs 철분 (Iron)**:
  * *기전*: 두 영양소는 소장 점막의 동일한 이가 금속 수송체(DMT-1)를 공유하므로, 동시에 존재할 경우 흡수율이 최대 50% 이상 상실됩니다.
  * *조치*: 철분은 공복(아침 식전), 칼슘은 식후(저녁 식후)로 배정합니다.
* **아연 (Zinc) vs 구리 (Copper)**:
  * *기전*: 고용량 아연의 지속 복용은 소장 내 메탈로티오네인 단백질을 과잉 유도하여 구리의 흡수를 차단, 구리 결핍성 빈혈을 유발합니다.
  * *조치*: 장기 복용 시 아연과 구리 복용 시간을 나누어 설계합니다.
* **마그네슘 (Magnesium) vs 칼슘 (Calcium) 고함량**:
  * *기전*: 칼슘의 함량이 마그네슘에 비해 지나치게 과도할 경우 마그네슘의 흡수를 물리적으로 억제합니다.
  * *조치*: 두 성분의 황금 비율(약 2:1) 유지를 권장하고, 고농도 단일제일 경우 아침과 저녁으로 나누어 복용하도록 안내합니다.

### ✨ 시너지 영양소 조합 기준 (Synergy)
보유 중인 영양 성분 중 아래 조합이 충족될 경우 대시보드 및 리포트에서 맞춤 시너지 카드를 표기하여 생체이용률을 극대화하도록 유도합니다.

* **철분 (Iron) + 비타민 C (Vitamin C)**:
  * *시너지 효과*: 비타민 C가 난흡수성 삼가철(Fe3+)을 흡수가 쉬운 이가철(Fe2+) 상태로 환원시켜 철분의 흡수율을 대폭 향상시킵니다.
* **칼슘 (Calcium) + 비타민 D (Vitamin D) + 비타민 K2**:
  * *시너지 효과*: 비타민 D가 장내 칼슘 흡수율을 높이고, 비타민 K2가 흡수된 칼슘을 혈관벽에 쌓이지 않게 뼈로 안전하게 인도(Osteocalcin 활성화)합니다.
* **CoQ10 + 오메가-3 (EPA/DHA)**:
  * *시너지 효과*: 지용성 항산화제인 CoQ10이 오메가-3의 지질 성분에 용해되어 소화관 내 흡수 효율이 극대화되며, 심혈관 건강 시너지를 냅니다.

## 4. 시간약리학(Chronopharmacology) 기반 복용 스케줄 배정 알고리즘

분석 엔진은 영양소의 활성 주기와 장 내부 환경(위산 농도 등)을 고려하여 복용 타임라인을 자동 분산 배치합니다.

1. **아침 공복 (기상 직후)**:
   * *배정 성분*: 철분(음식물 방해 극최소화), 유산균(위산이 분비되지 않은 공복 환경 생존 유도)
2. **아침 식후**:
   * *배정 성분*: 비타민 B군, 비타민 C (에너지 대사 활성화 및 위장 장애 유발 방지)
3. **점심 식후**:
   * *배정 성분*: 비타민 D, 비타민 A, 비타민 E, 오메가-3 (담즙산 분비가 많은 낮 식후 지용성 성분 흡수 최적화)
4. **저녁 식후 / 취침 전**:
   * *배정 성분*: 칼슘, 마그네슘 (칼슘의 뼈 흡수 극대화, 마그네슘의 부교감 신경 자극 및 수면 유도 기능 활용)
