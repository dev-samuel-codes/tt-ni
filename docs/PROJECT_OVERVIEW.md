# t+-ni — AI 영양제 분석 & 복약 관리 서비스

> 성분표 사진 한 장으로 모든 영양제의 중복·부족 성분을 분석해 드립니다.

---

## 소개

**t+-ni**는 AI 기반 영양제 분석 및 복약 관리 서비스입니다. 사용자가 영양제
라벨을 촬영하면, AI가 성분을 자동으로 추출하여 한국인 영양소 섭취기준(KDRIs)과
비교·분석하고, 시간약리학(Chronopharmacology) 원리에 기반한 최적의 복용 스케줄을
제안합니다.

마치 **주머니 속 AI 약사**처럼, 지금 복용 중인 모든 영양제를 한눈에 파악하고,
과잉·부족 성분은 없는지, 약물과의 상호작용은 없는지 종합적으로 관리할 수
있습니다.

---

## 핵심 기능

### 1. 영양제 라벨 촬영 & AI 성분 추출

| 항목            | 설명                                                                                                                                                                       |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **동작 방식**   | 영양제 라벨(원료명/함량 표기) 사진을 업로드하면, Vision AI가 이미지에서 텍스트를 읽고 성분명·함량·단위를 자동으로 추출합니다. iPhone HEIC 포맷도 자동으로 JPEG 변환됩니다. |
| **똑똑한 매칭** | 추출된 성분명은 한국인 영양소 데이터베이스의 표준 성분명으로 자동 매핑됩니다. 예를 들어 "아스코르브산"으로 적혀 있어도 "비타민 C"로 인식합니다.                            |
| **실시간 확인** | 분석 작업은 비동기로 처리되며, 작업 상태(대기 → 처리 중 → 완료/실패)를 실시간으로 확인할 수 있습니다. 추출 결과를 검토·수정한 후 최종 저장할 수 있습니다.                  |

### 2. 웹 검색 기반 영양제 정보 조회

직접 사진이 없더라도, 영양제 제품명만 입력하면 웹 검색을 통해 제품 정보를
수집하고, AI가 성분 데이터를 구조화하여 추출해 줍니다.

---

### 3. KDRIs 기반 영양소 분석 리포트

| 항목                   | 설명                                                                                                                                      |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **개인 맞춤 분석**     | 사용자의 성별, 출생연도(연령대), 임신/수유 여부에 따라 맞춤형 한국인 영양소 섭취기준(KDRIs)을 적용합니다.                                 |
| **통합 집계**          | 등록된 모든 활성 영양제의 성분을 하나로 합산하여, 1일 총 섭취량을 계산합니다.                                                             |
| **5단계 상태 판정**    | 각 영양소별로 **과잉**(UL 초과) / **주의**(UL의 80% 이상) / **부족**(RDA의 50% 미만) / **적정**(RDA 충족) / **미확인** 상태를 진단합니다. |
| **약물 상호작용 감지** | 복용 중인 처방약과 영양제 성분 간의 알려진 상호작용을 함께 감지해 경고합니다.                                                             |
| **시각화**             | 대시보드와 분석 리포트 페이지에서 요약 카드, 영양소별 상세 표, Recharts 막대 차트로 결과를 직관적으로 보여줍니다.                         |

---

### 4. 시간약리학 기반 복용 스케줄러

| 항목              | 설명                                                                                                                                   |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| **과학적 근거**   | 시간약리학(Chronopharmacology) — 약물의 효과가 복용 시간대에 따라 달라질 수 있다는 원리 — 에 기반하여 최적의 복용 시간대를 결정합니다. |
| **시간대 분할**   | 아침(공복), 식사와 함께, 저녁 등 시간대별로 영양제를 배치합니다.                                                                       |
| **상호작용 고려** | 영양제 간 상호작용(중대/경고/주의), 약물-영양제 간 상호작용(복용 금지/시간 분리/필수 권장)까지 종합적으로 반영합니다.                  |
| **복용 팁**       | 복용 시 주의사항이나 꿀팁도 함께 제공합니다.                                                                                           |

---

### 5. AI 채팅 상담

사용자의 프로필, 건강 상태, 복용 중인 약물, 등록된 영양제, 최신 분석 리포트까지
고려하여 **맥락에 맞는 개인화된 답변**을 제공하는 AI 채팅입니다.

- 대화 세션 관리 (생성/전환/삭제)
- 실시간 SSE 스트리밍 응답
- 대화가 길어지면 자동 요약
- 자주 묻는 질문 추천

---

### 6. 온보딩 & 프로필 관리

3단계로 구성된 온보딩을 통해 영양 분석에 필요한 사용자 정보를 수집합니다.

| 단계       | 수집 정보                                                          |
| ---------- | ------------------------------------------------------------------ |
| **Step 1** | 성별, 출생연도, 키, 몸무게, 임신/수유 여부                         |
| **Step 2** | 건강 상태 (고혈압, 당뇨 등 15개 프리셋 + 직접 입력), 알레르기 정보 |
| **Step 3** | 복용 중인 처방약 (이름 + 용법)                                     |

## 기술적 설계 원칙

- **개인 맞춤형 분석**: 사용자 프로필(연령·성별·건강 상태)을 기반으로 KDRIs
  참조값을 다르게 적용하여, 일률적인 기준이 아닌 개인에게 최적화된 결과를
  제공합니다.
- **비동기 + 실시간**: 무거운 AI 작업(이미지 분석, 영양소 리포트 생성)은
  백그라운드에서 비동기로 처리하고, 처리 상태를 실시간으로 구독하여 사용자
  경험을 해치지 않습니다.
- **사용량 제한**: 과도한 API 호출을 방지하기 위해 기능별·사용자별 일일 한도를
  설정하여 안정적인 서비스 운영을 보장합니다.
- **의료 면책**: 모든 분석 결과는 참고용이며, 의학적 진단이나 처방을 대체하지
  않음을 명시합니다.
