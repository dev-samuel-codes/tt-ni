# tt-ni 개인정보 처리방침

**최종 업데이트: 2026년 5월 20일**

tt-ni(이하 '서비스')는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」 등 관련 법령을 준수하고 있습니다. 본 개인정보 처리방침은 서비스가 수집하는 개인정보의 항목, 수집 목적, 보유 기간 등을 안내합니다.

---

## 제1조 (개인정보의 수집 항목 및 방법)

### 1. 수집 항목

| 구분 | 수집 항목 | 수집 시점 |
|------|-----------|-----------|
| **필수** | 이메일 주소, 이름(닉네임), 프로필 이미지 | 회원가입 (소셜 로그인) |
| **선택** | 성별, 출생연도, 키, 몸무게 | 프로필 설정 |
| **선택** | 건강 정보 (기저질환, 알레르기, 식이제한, 임신/수유 상태) | 프로필 설정 |
| **선택** | 복용 약물 정보 (약명, 복용 목적, 복용 빈도) | 프로필 설정 |
| **선택** | 영양제 라벨 이미지 | 영양제 등록 (사진 촬영/업로드) |
| **자동 수집** | 서비스 이용 기록, API 호출 기록, 기기 정보 | 서비스 이용 과정 |

### 2. 수집 방법
- 소셜 로그인 (Google, GitHub)을 통한 회원가입 시 자동 수집
- 프로필 설정 화면에서 이용자가 직접 입력
- 영양제 등록 시 이미지 업로드를 통한 수집
- 서비스 이용 과정에서 자동 생성·수집

---

## 제2조 (개인정보의 수집 및 이용 목적)

수집된 개인정보는 다음의 목적에 활용됩니다.

| 목적 | 상세 내용 |
|------|-----------|
| **회원 관리** | 본인 확인, 회원 식별, 서비스 이용 기록 관리 |
| **영양 분석 서비스** | 개인 맞춤형 영양소 분석, KDRIs 기준 비교, 과다/부족 분석 |
| **약물 상호작용 검사** | 복용 약물과 영양제 간 상호작용 분석 |
| **복용 스케줄링** | 시간약리학 기반 최적 복용 시간 설계 |
| **AI 채팅 상담** | 프로필·영양제·분석 결과를 반영한 개인화된 상담 |
| **API 사용량 관리** | 일일 API 호출 한도 관리 및 서비스 품질 유지 |
| **서비스 개선** | 이용 패턴 분석을 통한 서비스 품질 개선 |

---

## 제3조 (개인정보의 보유 및 이용 기간)

| 보유 정보 | 보유 기간 | 근거 |
|-----------|-----------|------|
| 회원 정보 (이메일, 이름) | 회원 탈퇴 시까지 | 서비스 이용계약 |
| 프로필 정보 | 회원 탈퇴 시까지 | 서비스 이용계약 |
| 영양제 등록 데이터 | 회원 탈퇴 시까지 또는 삭제 요청 시 | 서비스 이용계약 |
| 분석 리포트 | 회원 탈퇴 시까지 | 서비스 이용계약 |
| API 사용량 기록 | 수집일로부터 90일 | 서비스 운영 |
| 서비스 이용 로그 | 3개월 | 통신비밀보호법 |

단, 관계 법령의 규정에 의하여 보존할 필요가 있는 경우 아래와 같이 일정 기간 보관합니다.

| 보존 항목 | 보존 기간 | 근거 법령 |
|-----------|-----------|-----------|
| 계약 또는 청약철회 등에 관한 기록 | 5년 | 전자상거래법 |
| 소비자의 불만 또는 분쟁처리에 관한 기록 | 3년 | 전자상거래법 |

---

## 제4조 (개인정보의 처리 위탁)

서비스는 원활한 개인정보 업무처리를 위하여 다음과 같이 개인정보 처리업무를 위탁하고 있습니다.

| 수탁자 | 위탁 업무 |
|--------|-----------|
| Supabase, Inc. | 데이터베이스 호스팅, 인증 서비스, 스토리지 |
| Vercel, Inc. | 프론트엔드 호스팅, CDN |
| OpenAI, LLM | AI 이미지 분석, 영양성분 정제, 채팅 응답 생성 |
| Exa, Inc. | 영양제 제품 웹 검색 |

## 제5조 (개인정보의 국외 이전)

서비스는 다음과 같이 국외의 제3자에게 개인정보를 이전하고 있습니다.

| 이전받는 자 | 이전 국가 | 이전 항목 | 이전 목적 | 보유 기간 |
|-------------|-----------|-----------|-----------|-----------|
| Supabase, Inc. | 미국 | 서비스 이용 시 수집되는 모든 개인정보 | 데이터 저장 및 서비스 운영 | 회원 탈퇴 시까지 |
| Vercel, Inc. | 미국 | 서비스 이용 기록 | 프론트엔드 호스팅 | 회원 탈퇴 시까지 |
| OpenAI, LLM | 미국 | 영양제 라벨 이미지, 성분 정보 | AI 분석 처리 | 처리 후 즉시 삭제 |
| Exa, Inc. | 미국 | 영양제 검색 쿼리 | 제품 검색 | 처리 후 즉시 삭제 |

## 제6조 (이용자의 권리와 의무)

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

1. **개인정보 열람 요구**: 자신이 제공한 개인정보의 처리 현황을 열람할 수 있습니다.
2. **오류 등이 있을 경우 정정·삭제 요구**: 개인정보에 오류가 있거나 불필요한 경우 정정·삭제를 요구할 수 있습니다.
3. **처리정지 요구**: 개인정보의 처리를 정지하도록 요구할 수 있습니다.
4. **회원 탈퇴**: 서비스 내 설정 화면에서 언제든지 탈퇴할 수 있으며, 탈퇴 시 개인정보는 지체없이 파기됩니다.

권리 행사는 서비스 내 설정 화면 또는 개인정보 보호책임자에게 서면, 이메일 등으로 연락하시면 지체없이 조치하겠습니다.

---

## 제7조 (개인정보의 파기)

1. 서비스는 개인정보 보유 기간의 경과, 처리 목적 달성 등 개인정보가 불필요하게 되었을 때에는 지체없이 해당 개인정보를 파기합니다.
2. 전자적 파일 형태의 정보는 복구 및 재생이 불가능한 방법으로 영구 삭제합니다.
3. 이미지 파일은 Storage에서 삭제하며, 데이터베이스 기록은 회원 탈퇴 시 일괄 삭제됩니다.

---

## 제8조 (개인정보의 안전성 확보 조치)

서비스는 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다.

1. **접근 통제**: Row Level Security(RLS)를 통한 데이터 접근 제한
2. **암호화**: 데이터 전송 시 TLS 암호화 적용
3. **접근 기록 관리**: 개인정보 접근 기록을 logging
4. **보안 프로그램**: 주기적인 보안 점검 및 취약점 수정

---

## 제9조 (민감 정보 처리에 대한 안내)

본 서비스는 건강 관련 민감 정보를 처리합니다.

1. **수집하는 민감 정보**: 기저질환, 알레르기, 임신/수유 상태, 복용 약물
2. **수집 목적**: 개인 맞춤형 영양 분석 및 약물 상호작용 검사
3. **동의 거부 권리**: 민감 정보 제공을 거부할 수 있으며, 이 경우 맞춤형 분석 기능이 제한될 수 있습니다.
4. **중요 안내**: 본 서비스의 분석 결과는 참고용이며, 의학적 진단이나 처방을 대체하지 않습니다. 건강 관련 결정은 반드시 전문의와 상의하시기 바랍니다.

---

## 제10조 (개인정보 보호책임자)

개인정보 처리에 관한 업무를 총괄해서 책임지고, 개인정보 처리와 관련한 정보주체의 불만처리 및 피해구제를 위하여 아래와 같이 개인정보 보호책임자를 지정하고 있습니다.

| 구분 | 내용 |
|------|------|
| **직책** | 개인정보 보호책임자 |
| **이메일** | (추후 기입) |

## 제11조 (개인정보 처리방침의 변경)

본 개인정보 처리방침은 시행일로부터 적용되며, 법령 및 방침에 따른 변경내용의 추가, 삭제 및 정정이 있는 경우 변경 사항의 시행 7일 전부터 서비스 내 공지사항을 통하여 고지할 것입니다.
